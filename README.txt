##Dice Game

A simple console-based dice game in C#.

This is a console-based dice game where the player competes against the computer in a series of rounds. Each round involves rolling a six-sided die for both the player and the computer. The winner of each round is determined by the higher roll. The game consists of 10 rounds, and the player with the most wins at the end is declared the overall winner.

How to Play

Run the program.
Press any key to roll the dice for each round.
The program will display the rolls and announce the winner of each round.
After 10 rounds, the program will announce the overall winner.
